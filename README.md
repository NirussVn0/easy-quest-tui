Automatically completes all Discord Quests for you

> ⚠️ Selfbots violate Discord ToS. Use at your own risk.

![demo](screenshot/preview.png)

## ⚙️ Install

Requires [Node.js](https://nodejs.org/) v18+.

```bash
git clone https://github.com/Kitosium/lazy-quest.git
cd lazy-quest
npm install
```

## 🔑 Setup

Copy `.env.example` to `.env` and fill in your token:

```
TOKEN=your_discord_token
```

## 🚀 Run

Run

```bash
npm start
```
